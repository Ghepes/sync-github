Universal Satellite Search

This example demonstrates how to embed the Satellite Search in any website, with CORS-free operation.

How to Use This Integration

Step 1: Add the container to your HTML

<div id="satellite-search"></div>

Step 2: Include the script

<script src="universal-search.js"></script>

Or you can include it directly from your CDN:

<script src="https://search.wromo.com/examples/universal-search.js"></script>

Step 3: Configure (optional)

You can customize the search by adding this before the script:

<script>
  window.satelliteSearchConfig = {
    apiUrl: "https://search.wromo.com/api",
    displayMode: "redirect"
  };
</script>

Why This Works Everywhere

This integration uses multiple techniques to fetch data:

  1. JSONP - Works across domains without CORS issues
  2. Standard CORS - Used as a fallback if JSONP fails
  3. CORS Proxy - Used as a last resort if both methods fail

It also includes better error handling and fallbacks to ensure a good user experience even when network issues occur.
