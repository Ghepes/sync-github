This is an example of how to embed Satellite Search in any website.

About This Example

This example demonstrates how to embed the Satellite Search functionality in any HTML website. The search is powered by a decoupled backend that uses Google Programmable Search Engine (CSE).

Note About Embedded Mode

Some websites (like AliExpress, Amazon, etc.) prevent being displayed in iframes for security reasons. When you click on these results, you'll see an option to open them in a new tab instead.

If you prefer to always open results in a new tab, you can set displayMode: "redirect" in the configuration.

Important: Before using this example, make sure to update the apiUrl in the satellite-search.js file to point to your deployed API.
