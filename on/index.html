Embed Mode

This example demonstrates how to embed search results within your page using iframes.

Features:

  • Keeps users on your website
  • Displays content in a modal overlay
  • Gracefully falls back to new tabs for sites that block embedding
  • Provides a seamless user experience
View Embed Example

Redirect Mode

This example demonstrates how to open search results in new tabs, which works with all websites.

Features:

  • Compatible with all websites, including those that block iframes
  • Simpler implementation with no iframe handling
  • Clear visual indication that links open in new tabs
  • Better for sites where most results come from iframe-blocking domains
