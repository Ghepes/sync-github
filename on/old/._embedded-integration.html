About This Example

This example demonstrates how to embed the Satellite Search functionality in any HTML website. The search is powered by a decoupled backend that uses Google Programmable Search Engine (CSE).

You can customize the appearance and behavior of the search by modifying the configuration in the satellite-search.js file.

Important: Before using this example, make sure to update the apiUrl in the satellite-search.js file to point to your deployed API.
