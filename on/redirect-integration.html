About This Example

This example demonstrates the "redirect" mode of Satellite Search, which always opens search results in a new tab. This is ideal for sites that don't allow embedding or when you prefer a simpler user experience.

Redirect Mode Features

  • All search results open in a new browser tab
  • Works with all websites, including those that block iframe embedding
  • Simpler implementation with no iframe handling needed
  • Small external link icon indicates links open in a new tab

Comparison of Display Modes

Embed Mode

Attempts to display the target website within an iframe on your page.

Pros: Keeps users on your site, provides a seamless experience

Cons: Many sites block embedding with X-Frame-Options headers

View Embed Mode Example

Redirect Mode

Always opens search results in a new browser tab.

Pros: Works with all websites, simpler implementation

Cons: Takes users away from your site
